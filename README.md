## üì• Quick Start (Windows Beta)

1.  **Download**: Get the latest installer (`.exe`) from the **[Releases page](https://github.com/csheng21/operatorX02/releases/tag/v1.0.0-beta)**.
2.  **Run**: Execute the `.msi` installer.
3.  **Launch**: Open Operator X02 from your Start Menu.
4.  **Configure AI**:
    *   Bring your own API key (for OpenAI, Anthropic, etc.) in Settings.
    *   Or set up a local model via Ollama for 100% offline use.
5.  **Open a Project**: Start coding with your new AI team!

**Requirements**:
*   Windows 10/11 (64-bit)
*   WebView2 Runtime (usually pre-installed)

## ‚ùì FAQ

**Q: Is this really free?**  
A: Yes! Operator X02 is and will always be free. It's open-source under the MIT license.

**Q: Where is my data stored?**  
A: **Entirely on your local machine**. No cloud, no telemetry, no tracking.

**Q: Can I use it offline?**  
A: Absolutely. Combine it with a local model (like Ollama) for a 100% offline, air-gapped experience.

**Q: When will macOS/Linux versions be available?**  
A: Planned after the March 15 full release. Stay tuned!

**Q: The beta is only an `.exe`. Where is the source code?**
